E-Commerce Revenue Dashboard

This project is a Power BI dashboard built to analyze and visualize e-commerce revenue performance over time. It provides a quick overview of sales trends, customer behavior, and operational insights to support better business decisions.

Dashboard Highlights

  Key Metrics
    
    Total Revenue
    Total Transactions
    Average Daily Transactions
    Total Products Sold

  Interactive Filters
  
    Date range selector
    Gender (Male / Female)
    Customer Login Type

  Revenue Breakdown
  
    By transaction status (Success vs Failed)
    By device type (Web vs Mobile)
    By city
    By product category
    By delivery type (One-day vs Normal)

  Trend Analysis
  
    Daily revenue trend to identify peaks, drops, and seasonal patterns

  Tools Used

    Power BI – Data modeling, DAX measures, and visualizations
    Power Query – Data cleaning and transformation

Purpose

  The goal of this dashboard is to present complex e-commerce data in a clean and easy-to-understand format. It helps stakeholders quickly identify where revenue is coming from, how customers are purchasing, and which categories or cities are performing best.
